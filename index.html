California Speed Game

This is an app for playing the card game California Speed, which incorporates drag-and-drop functionality for moving the cards into piles. We also used sockets to let two users on the same network play the game against each other. Once the game finishes, the players are each asked for their names, then that game's information is stored in a MongoDB database and displayed for the users to see.

    My Contributions:
  • The drag-and-drop functionality and display for the cards and piles
  • All of the sockets functionality
  • The game history page
  • Some of the game logic, including the card shuffling algorithm

One of the technical challenges we overcame during this project was differentiating between the players, which we solved by using each player's socket id in a key value pair with their player number.
