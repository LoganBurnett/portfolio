Hangman App

This is an app to play the game of Hangman. The user enters their name, then a random word is chosen from the database and its length is sent back and displayed for them to start guessing letters. Once the game is finished, their score is put into the Mongo database and the high scores for each word length are displayed.

    My Contributions:
  • The backend route of getting a new random word
  • The backend route of guessing a letter
  • The backend route of ending the game and storing the result
  • Helped with properly displaying the word length and position of correct letters

Number Guessing Game

This is a simple number guessing game that was made to learn server-side components with Express. The app prompts the user for a name and generates a random number between 1 and 100 on the backend. The user then submits guesses until they are correct, then their score is stored in a Mongo database, which is then displayed in a high scores table.
